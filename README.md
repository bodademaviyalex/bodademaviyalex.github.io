# Invitación de boda · Alejandro & Maravillas

Invitación digital para la boda del 13 de junio de 2026 en Lorca.

## Stack

- HTML + CSS + JS vanilla, un solo archivo (`index.html`)
- Imágenes optimizadas en `assets/`
- Formulario RSVP vía [Formsubmit](https://formsubmit.co) (envía a email)
- Hospedaje: GitHub Pages

## Publicación

Se publica automáticamente en GitHub Pages desde la rama `main`.

URL: https://bodademaviyalex.github.io/

## Personalización

- **Teléfonos, IBAN, ubicaciones**: editar directamente en `index.html`
- **Email de destino del RSVP**: cambiar el `action` del `<form>` (buscar `formsubmit.co`)
- **Imágenes**: reemplazar en `assets/` manteniendo los nombres de archivo
